# 云服务器折扣信息聚合网站

## 项目简介
云服务器折扣信息聚合网站是一个专门收集和展示各大云服务商优惠信息的平台，包括阿里云、腾讯云、华为云、UCloud等主流云服务商的折扣活动、优惠券信息。

## 功能特性

### 前端功能
- **首页轮播图**: 展示各大云服务商的最新优惠活动
- **云服务商分类**: 按阿里云、腾讯云、华为云、UCloud分类展示
- **折扣信息汇总**: 集中展示所有云服务商的优惠信息
- **搜索筛选功能**: 支持按关键词和云服务商筛选折扣信息
- **响应式设计**: 适配各种设备屏幕尺寸
- **SEO友好**: 包含适当的meta标签和结构化数据

### 后台管理功能
- **添加折扣信息**: 通过表单添加新的云服务器折扣信息
- **编辑折扣信息**: 修改已有的折扣信息
- **删除折扣信息**: 删除过期或无效的折扣信息
- **数据管理**: 集中管理所有折扣数据

## 技术架构
- **前端**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **后端**: Node.js + Express.js
- **数据存储**: 前端模拟数据（实际部署中可替换为数据库）

## 安装部署

### 本地运行
1. 安装Node.js (版本 >= 14.0.0)
2. 克隆或下载项目文件
3. 在项目根目录执行：
```bash
npm install
npm start
```
4. 访问 `http://localhost:3000`

### 生产环境部署
1. 将项目文件上传到服务器
2. 安装依赖：`npm install --production`
3. 启动服务：`npm start`
4. 配置反向代理（如Nginx）指向3000端口
